# 要件定義書 - enockey.com

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| サイト名 | enockey.com |
| オーナー | Daisuke Enoki |
| 目的 | 写真・イラスト・音楽・エンジニアリングを通じて世界観を表現する個人サイト |
| ターゲット | サイトを訪れた人全般 |
| 与えたい印象 | 優しさ・美しさ・かっこよさが共存している人間の作品集 |

---

## ゴール

訪れた人が「この人の世界観が好きだ」と感じる、ポートフォリオ兼名刺サイトを作る。

---

## 機能要件

### 必須機能（実装済み）

| # | 機能 | 説明 |
|---|------|------|
| F1 | トップページ | 世界観を伝えるランディングページ |
| F2 | プロフィール | 自己紹介・各セクションへの導線 |
| F3 | 写真ギャラリー | プロジェクト単位で写真を閲覧できる |
| F4 | イラストギャラリー | イラスト作品をグリッド表示 |
| F5 | お問い合わせフォーム | 名前・メール・メッセージを送信できる |
| F6 | スパム対策 | ハニーポット・送信時間チェック |

### 今後追加予定

| # | 機能 | 優先度 |
|---|------|--------|
| F7 | 音楽ページ | 中 |
| F8 | ブログ・記事ページ | 低 |
| F9 | 独自ドメインのメール送信 | 高 |

---

## 非機能要件

| # | 要件 | 現状 |
|---|------|------|
| N1 | 高速な表示 | Next.js SSR・Cloudinary CDN で対応済み |
| N2 | スマホ対応（レスポンシブ） | Tailwind CSS で対応済み |
| N3 | ダークモード禁止 | globals.css で固定済み |
| N4 | セキュリティ | APIキーはサーバー側のみ・バリデーションあり |
| N5 | 無料〜低コスト運用 | Vercel 無料枠・Cloudinary 無料枠で運用 |
